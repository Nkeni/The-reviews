The position property

The position property specifies the type of positioning method used for an element.

5 different position values

  • Static
  • Relative
  • Fixed
  • Absolute
  • Sticky

Elements are positioned using top, bottom, left and right properties after the position property is set first. Otherwise it wont work.

Position: Static

By default, HTML elements are positioned static. They are not affected by the top, bottom, left and right properties. They are always positioned according to the normal flow of the page.

This div element has position: static;

Position: Relative

An element with this position is positioned relative to its normal position

Setting the relatively position element to top, bottom, left and right will cause an element to be adjusted to specified direction away from its normal position.

This div element has position: relative;

Position: Fixed

An element is positioned relative to its viewport. It always stays in the same place even when you scroll the page. The top, bottom, right and left positions can be used.

This div element has position: fixed;

Position: Absolute

An element is positioned relative to the nearest positioned ancestor instead of viewport

if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.

Absolute positioned elements are removed from the normal flow, and can overlap elements.

This div element has position: relative;
This div element has position: absolute;

Position: Sticky

An element with position: sticky; is positioned based on the user's scroll position.
