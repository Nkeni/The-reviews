Z-index

Z-index specifies the stack order of an element.

When elements are positioned, they can overlap other elements, so the Z-index property specifies the stack order of an element. It specifies which element should be placed infront or behind others.

An element can have a NEGATIVE or POSITIVE stack order.

z-index only works on positioned elements (position: absolute, position: relative, position: fixed, or position: sticky) and flex items (elements that are direct children of display: flex elements).
