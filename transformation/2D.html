Transformation property

The transform property applies a 2D or 3D transformation to an element. It allows you to ROTATE, SCALE, MOVE SKEW elements

CSS 2D TRANSFORMS METHODS

With the transform property you can use the following 2D transformation methods;

  • translate()
  • rotate()
  • scaleX()
  • scaleY()
  • scale()
  • skewX()
  • skewY()
  • skew()
  • matrix()

The translate method

The translate method moves an element from its current position according to the values given for the X-axis and Y-axis

green

The rotate method

The rotate method rotates an element clockwise or anti clockwise according to the given degree

POSITIVE value rotates the element clockwise

blue

The NEGATIVE degree value will rotate the element anticlockwise

plum

The scale method

The scale method increases or decreases the size of an element according to the parameters given for the width and height.

red

The scaleX method

The scaleX method increases or decreases the width of an element

pink

The scaleY method

The scaleY method increases or decreases the height of an element

The skew () method

The skew() method skews an element along the X and Y-axis by the given angles. The first value is for X-axis and the second is for Y-axis

The skewX method

The skewX() method skews an element along the X-axis by the given angle.The value is defined in degrees

green

The skewY method

The skewY() method skews an element along the Y-axis by the given angle.The value is defined in degrees
